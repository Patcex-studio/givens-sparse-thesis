# Experiments

This folder contains executable experiments that complement the theoretical development in the main repository.

## Scripts

- `run_random_uccsd_like.py` — Generate and simulate random UCCSD-like Givens rotations on a sparse state.

## Example

```bash
python experiments/run_random_uccsd_like.py --n-qubits 148 --m 1000
```

## Results

The file `results/random_uccsd_148q_1000.txt` records the output of the default experiment run.
